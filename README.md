# Hidden Markov Model

the implementation of hidden markov model with C++

## Environment

- C++ 11+
- gcc 8.0+
- boost 1.71.0
- glog 0.4.0
- gflag 2.2.2

## Note

- glog: **we use gcc for building glog**, so you must use gcc to build
- gflag: **we use gcc for building glog**, so you must use gcc to build
